# [Artalk](https://artalk.js.org)

![banner-2021.png](https://bu.dusays.com/2021/07/16/8ea9ca8418e55.png)

> 一款简洁有趣的自托管评论系统

{% note color:red 官方文档：[Artalk (v2)](https://artalk.js.org/)，本文档做存档处理，不再更新。历史悠久，已从项目页面下架 %}



## 特性

- 有趣有爱

- 自托管

- 支持Markdown

- 表情自定

- 滑稽表情包

- 管理员密码，防冒名

- 验证码，提交频率限制

- 通知中心，邮件提醒

- 仅管理员可评论

- 无限层级回复

- 滚动加载更多

- 内容自动保存

- 暗黑模式

- 评论折叠

- 一页多个评论

- TypeScript
