## 物质表达方式

{% note 等待修改 color:yellow %}

| CN           | MEAN |
| ------------ | ----- |
| 分子式       | 用元素符号和下标表示分子中每种元素的原子数 |
| 结构式       | 用化学键连接的方式显示分子中每个原子的位置和连接方式 |
| 结构简式 | 简化的结构式，通常省略碳原子和氢原子之间的连接线 |
| 实验式/最简式 | 元素的最简整数比 |
| 键线式 | 用线段表示共价键或离子键 |
